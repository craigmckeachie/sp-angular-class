A simple directive
<simple-greeting></simple-greeting>

A directive which takes an @ argument:
<at-greeting name="John"></at-greeting>

What if we want to pass something from the controller?
<at-greeting name="$ctrl.person.name"></at-greeting>

That didn't work, let's try
<at-greeting name="{{ $ctrl.person.name }}"></at-greeting>

Somewhat unsatisfying. Let's use = to pass in a value
<equal-greeting name="$ctrl.person.name"></equal-greeting>

Heck, we could even pass in a complete object!
<object-greeting obj="$ctrl.person"></object-greeting>

Sometimes, you want to pass in some code to execute, which is what & is for. Here, we will pass in the callMe() function. Click on the content below to activate it.

<event-greeting on-click="$ctrl.callMe()"></event-greeting>
